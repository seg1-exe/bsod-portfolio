ARTEFACT

a creative 3D website and an e-commerce website

For the release of ARTEFACT's visuals, leap contacted me to create two websites. The first highlights the 3D scene created for the trailer and allows users to explore the 3D versions interactively. And the second is an e-commerce site for selling these CDs.

Using the 3D scene created by Axel Coudray, I enabled users to scroll through the cave and view the two CDs. I also added small rotation animations that activate when clicked.

For this e-commerce site, I used the same style I had created for the site's 3D cartels, a kind of fairly subtle glassmorphism.
